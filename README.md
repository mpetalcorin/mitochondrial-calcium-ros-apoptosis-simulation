# Mitochondrial Calcium, ROS, and Apoptosis Simulation

A biologically informed, synthetic single-cell simulation workflow for exploring how mitochondrial calcium loading, reactive oxygen species, membrane potential, ATP production, permeability transition, cytochrome c release, caspase activation, and apoptotic commitment interact across multiple cellular conditions.

## Overview

This repository contains a Jupyter notebook that simulates literature-consistent mitochondrial stress biology and performs downstream analysis, including summary statistics, correlation analysis, principal component analysis, apoptosis classification, feature-importance ranking, and resilient-versus-vulnerable cell comparisons.

The workflow is designed for reproducible computational exploration, method prototyping, figure generation, and educational use. It is not intended for clinical use or for claiming experimentally validated biological effects without independent laboratory confirmation.

## Scientific scope

The simulation encodes several biologically plausible principles from mitochondrial physiology:

- Moderate mitochondrial calcium can support ATP production.
- Excess mitochondrial calcium can promote dysfunction and apoptotic signaling.
- ROS can amplify mitochondrial stress.
- Combined calcium overload and oxidative burden can increase apoptotic commitment.
- Reduced oxidative load or reduced mitochondrial calcium uptake can partially rescue stressed states.

All outputs are synthetic and reported in normalized units.

## Simulated conditions

The default notebook simulates seven conditions:

1. Control  
2. Mild Stress  
3. Ca Overload  
4. ROS Stress  
5. Combined Ca and ROS Stress  
6. Antioxidant Rescue  
7. MCU Inhibition

## Key outputs

Running the notebook generates:

- A synthetic single-cell dataset.
- Condition-level summary statistics.
- Bar plots of mitochondrial stress features.
- Apoptosis frequency plots.
- Correlation heatmaps and scatter plots.
- Principal component analysis, PCA.
- Logistic regression and random forest models for apoptosis prediction.
- Feature-importance analysis.
- Integrated stress-score rankings.
- Exported CSV files for downstream analysis.

4. Review the generated tables, plots, and exported CSV files.

## Out-of-scope uses

This repository should not be used for:

- Clinical diagnosis.
- Patient risk prediction.
- Therapeutic decision-making.
- Claims of experimentally validated biological effect without wet-lab confirmation.
- Estimation of absolute biological effect sizes from synthetic outputs.

## Reproducibility

The workflow uses fixed random seeds to improve reproducibility. Because the dataset is simulated, changes to parameters, conditions, or model structure will change the resulting distributions.

## Limitations

- The dataset is synthetic, not experimentally measured.
- Variables are normalized rather than tied to one assay platform.
- Temporal dynamics are simplified.
- Cell-type-specific behavior is not explicitly modeled.
- Biological pathways outside the encoded framework are omitted.
